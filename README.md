# E-Commerce Data Interaction Platform

A full-stack analytics application built with **Flask**, **React**, and **MySQL** for real-time data management, querying, and interactive reporting.

## Features

- Full-stack architecture with **Flask backend** and **React frontend**
- **MySQL integration** for data storage, retrieval, and query execution
- Advanced SQL analytics:
  - CRUD operations
  - Set operations: `UNION`, `INTERSECT`, `EXCEPT`
  - Window functions: `NTILE`, `CUME_DIST`, `ROLLUP`
- Interactive dashboards for Exploratory Data Analysis (EDA)
- Generates actionable insights from large e-commerce datasets

## Tech Stack

- **Backend:** Flask, Python  
- **Frontend:** React, JavaScript  
- **Database:** MySQL  
- **Visualization:** React Charts / Custom Dashboards
